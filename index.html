Post #2: HTML and CSS, Classes and IDs

Two more weeks with Dev Bootcamp, and we already have our own websites! We've learned a lot about HTML and CSS and how to use them together to begin to create websites like the ones we see every day. It's pretty awesome that we've gotten through so much material in such a short time. I know we've barely scratched the surface, but to think that I didn't know any of this a couple weeks ago really excites me. We learned about a few different tools and concepts in CSS this week such as the difference between margins, borders and padding, different types of positioning, display properties, and how to use classes and ids. I'm going to be talking about classes and ids.
